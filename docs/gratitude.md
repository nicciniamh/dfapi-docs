# Gratitude

A number of Free and Open-Source Software (FOSS) projects have directly made this system possible.

I wish to extend gratitude in general and in specific to persons and organizations without whom this project would have not been possible. In particular, thanks to the tens of thousands of volunteers on FOSS projects who put their resources and time away from their jobs, lives, and family working, often, thanklessly and uncompensated, to help fulfil the dream of software and tools for everyone.

* Linux Torvalds for creating Linux (along with thousands of other developers)
* [Guido van Rossum](https://gvanrossum.github.io/) - Creator of and BDFL (Benevolent Dictator For Life) of Python
* [The Python Software Foundation](https://www.python.org/psf-landing)
* [Nikolay Kim and Andrew Svetlov](https://github.com/aio-libs) mainers of aiohttp.
* [MongoDB Inc.](https://www.mongodb.com) for their fabulous database I love to hate. (Mostly love)
* Microsoft - for Github and Visual Studio Code.
* [The Debian Foundation](https://www.debian.org) who has pushed the idea of Software for Everyone.
